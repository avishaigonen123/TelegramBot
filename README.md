# TelegramBot
telegram bot translate and forward

simple program that use my user in telegram (this isn't a bot), 
this translate messages from a secret channel (https://t.me/SamaNewsAgency), 
and forward this into my channel :)

Here you enter to get the api data of your user.
https://my.telegram.org/auth?to=apps

If you want to use the batch files easing your process of runing, you need to download auto-py-to-exe:
pip install auto-py-to-exe
and then, changing the directories in the runBot.bat file and also in the killBot.bot. 
notice also to change the in the "cd" line in the runBot.bat the path of your exe file, where the session is being created.
